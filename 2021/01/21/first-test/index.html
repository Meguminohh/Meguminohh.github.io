Github Page页面无法访问?

参考网上找的博客搭建教程https://www.cnblogs.com/shwee/p/11421156.html

按步骤做到部署文章到github仓库后，github pages无法访问…

我的解决方式：

  1. 去IP地址查询网站查询自己博客的IP地址，如：xxx.github.io

    得到以下IP：
  2. 修改host文件，如：ip xxx.github.io

这样github pages就能成功访问了。
